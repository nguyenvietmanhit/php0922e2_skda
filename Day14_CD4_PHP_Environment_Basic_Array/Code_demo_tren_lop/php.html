1 - IDE Code PHPStorm: + Là công cụ code chuyên dành PHPStorm + Khá nặng + Phím tắt: Ctrl D -> nhân đôi dòng hiện tại Ctrl Shift N -> tìm theo tên file trong toàn project Alt F1 Project View -> xem vị trí file hiện tại trong cấu trúc project 2 - Ngôn ngữ lập trình PHP + Hypertext Preprocessor, là ngôn ngữ lập trình chuyên dùng để tạo ra ứng dụng web PHP cũng có thể viết API để các ứng dụng khác sử dụng (Android, IOS, Desktop ...) + Tạo ra web động - là web kết nối đc CSDL + Mã nguồn mở + Cộng đồng hỗ trợ lớn + Phần mở rộng là .php 3 - Môi trường phát triển Web với PHP: là điều kiện bắt buộc để website viết bằng PHP có thể chạy đc, 3 thành phần: + Webserver: máy chủ lưu trữ web trong 1 thư mục nào đó, có 2 phần mềm chính cài trên webserver là Apache và Nginx + PHP Engine: ngôn ngữ PHP + Database: cơ dữ dữ liệu (MySQL) dùng để lưu trữ dữ liệu (web động) -> cài 3 thành phần trên thủ công -> phức tạp, thay vào đó cài phần mềm tích hợp cả 3 thành phần trên: XAMPP, WAMPP ... 4 - Cách chạy file PHP với XAMPP: + Bắt buộc phải mở XAMPP, start 2 module Apache và MySQL + file .php bắt buộc phải đặt trong cái webserver của XAMPP: C:/xampp/htdocs - Tạo C:/xampp/htdocs/php0922e2_test/manhnv.php + Chạy file: http://localhost/php0922e2_test/manhnv.php
